VisualQA

Conventional handwriting recognition techniques like using a sliding window algorithm with a LeNet do not work efficiently on Devanagri (the script in which many Asian languages including Hindi, Bengali, Nepali, Marathi etc. are written) because of the similarities in alphabets like \ka and \pha. I implemented a software in Torch that first uses computer vision to determine the size of one line of text, and then uses a sliding window algorithm with a ResNet-152, trained on a self-generated dataset, that can recognise Hindi and Bengali characters with an accuracy of over 94%.
The software can therefore interpret entire pages of handwritten text in Devanagri and convert to UTF-8 characters with an unmatched accuracy
